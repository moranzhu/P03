Non-Interactive Elements

Headings

CSS Selectors

.mainheading, .subheading, .thirdheading

Sample Code

<h1 class="mainheading">HEADING H1</h1> <h2 class="subheading">Heading h2</h2> <h3 class="thirdheading">Heading h3</h3>

Rendered Element

HEADING H1

Heading h2

Heading h3

Paragraph

CSS Selectors

.paragraph

Sample Code

<p class="paragraph">Hello! My name is Moran Zhu.</p>

Rendered Element

Hello! My name is Moran Zhu. I am a passionate senior design student at Simon Fraser University. With my academic background, I have the knowledge and experiences in UX, UI and print design. I also dabble in programming, with intermediate level of proficiency of HTML/CSS and Java. I believe design is for people and their everyday lives, thus I am constantly observing and learning from the people and objects around me.

In-text emphasize

CSS Selectors

.emphasized

Sample Code

<em class="emphasized">important thing</em>

Rendered Element

This is an important thing.

Interactive Elements

CSS Selectors

.intext_link
